The linkage from InRoads v08.08.00.46 to MathCAD v14
========================================================================================== 
Operating System: Windows XP 
Prerequisite: InRoads v08.08.00.46 and MathCAD v14 have been correctly installed. 
For the software installation instruction of each, please find the HELP files embedded in the software or contact technical support of corresponding vendor.

File needed for this linkage
------------------------------------------------------------------------------------------
1. Alignment file "C01a 2088prop.alg"
2. MathCAD files "001-Highway.xmcd"

Linkage Test Run
------------------------------------------------------------------------------------------
1. Open InRoads v08.08.00.46 and load the alignment file "C01a 2088prop.alg";
2. In the window "geometry projects", choose the item "2088prop". Under this item, choose one of the six pairs of alignments;
3. Right click on one alignment, then choose "Review". Save all the information as text file;
4. Open MathCAD file "001-Hignway.xmcd", enter the text file name under item "Highway Inputs";
